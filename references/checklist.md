# Checklist: newsletter-writer Execution

## Phase 1: Research Planning
- [ ] Define scope and mandatory topics.
- [ ] Map research files.
- [ ] Create `plan-research.md` as per [template](./output-formats/plan-research.md).

## Phase 2: Research Execution
- [ ] Gather findings for each sub-topic.
- [ ] Populate `Research/` directory with research files as per generated `<newsletter-name>/plan-research.md` in Phase 1.

## Phase 3: Newsletter Planning
- [ ] Map research to episodes.
- [ ] Plan visuals.
- [ ] Create `plan-newsletter.md` as per [template](./output-formats/plan-newsletter.md).
- [ ] Create `00-introduction-to-newsletter.md` as per [template](./output-formats/introduction-to-newsletter.md).

## Phase 4: Newsletter Writing
- [ ] Draft all episodes.
- [ ] Generate final episode index and append it to previously generated `<newsletter-name>/Episodes/00-introduction-to-newsletter.md`.
- [ ] Verify against style guide.
